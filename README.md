## 테스트 방법

### 사용 스택
- node js
- express
- yarn(package manager)

### 구동 방법
- 패키지 설치
```
yarn
```
- 서버 시작
```
node app.js
```
- 8080 포트로 접속(GET)했을 시 Hello World!가 뜨면 성공입니다
